📚 Built a Comic Upload & Reader Platform using Django

I developed a web platform called **AnimeWorld** that allows users to upload, manage, and read comics directly in the browser.

🔹 Overview
The system supports uploading comics either as PDF files or as multiple images, making it easier for creators to publish their content in different formats.

🔹 Key Features
• Upload comics as PDF files or multiple images
• Automatic image-to-PDF conversion using Python
• Structured database models for users, categories, and comic metadata
• Secure user authentication for content ownership and upload tracking
• Validation logic to prevent invalid uploads (e.g., PDF + images simultaneously)

🔹 Tech Stack
Django | Python | Bootstrap | JavaScript | SQLite

🔹 Key Learnings
• Handling complex file uploads in Django
• Designing relational database models using Django ORM
• Managing file validation and storage efficiently

Building this project helped me understand how to design a structured backend system for managing media content and user-generated uploads.
